Credits and Legalese

Who's to Blame

App Tamer was conceived, designed, and programmed by Jon Gotow after he got tired of Safari chewing up his battery while he was sitting at a coffee shop. User interface design is the result of collaboration between Jon, his sons Ben and Andrew Gotow and Sanford Selznick.

Thanks to Andy Matuschak for providing the Sparkle framework to all Mac developers. App Tamer uses it to automatically check for, download and install updates. See <https://github.com/sparkle-project/Sparkle> for more information.

App Tamer's process monitoring code is based in part on AGProcess by Aram Greenman.

Detection of Apple Silicon CPU configurations is based on the Darwin module from the hwloc project, written by Brice Goglin and Jeff Squyres.

Some user interface elements incorporate BGHUDAppKit by Tim Davis.
