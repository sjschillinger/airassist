Installation

Installation d'App Tamer

App Tamer peut être placée n'importe où sur votre disque dur, mais nous vous recommandons de le placer dans votre dossier Applications aux côtés de Safari, Aperçu, Mail et de toutes les autres applications qui viennent avec votre Mac. Il suffit de faire glisser l'icône de l'application App Tamer dans votre dossier Applications. Double-cliquez ensuite sur App Tamer pour le lancer et l'explorer.

Désinstallation d'App Tamer

Quittez App Tamer depuis son menu dans votre barre des menus si elle est en cours d'exécution, puis faites glisser l'icône de l'application App Tamer dans la corbeille pour la désinstaller.

Quelques détails d’intérêts

La première fois que vous lancez App Tamer, on vous demandera votre mot de passe d'administrateur afin qu'il puisse installer son application auxiliaire. Cette application auxiliaire est utilisée pour obtenir des informations sur les applications en cours d'exécution sur votre ordinateur. Elle s'appelle com.stclairsoft.AppTamer.Helper et est placée à l'emplacement suivant :

/Library/PrivilegedHelperTools/com.stclairsoft.AppTamer.Helper

Cette application auxiliaire n'est exécutée que par App Tamer, donc si vous supprimez l'application App Tamer, elle ne sera jamais utilisée. Cependant, si vous souhaitez supprimer complètement toutes les traces d'App Tamer de votre système, vous devez également supprimer ce fichier.
