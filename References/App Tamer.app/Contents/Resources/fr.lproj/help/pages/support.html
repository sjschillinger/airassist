Assistance technique

Si vous souhaitez nous faire part de problèmes, de réclamations, d'éloges, de suggestions, veuillez nous le faire savoir. Cependant, si vous souhaitez une réponse à une question, veuillez consulter les ressources disponibles avant de nous contacter. Un certain nombre de sources utiles sont listées ici.

Consulter la documentation

Oui, vous êtes en train de lire ceci, aussi vous avez probablement consulté la documentation pour savoir si elle répondait à votre question. Néanmoins, effectuez une analyse rapide de plus pour voir si l'information est ici. Il y a de bonnes chances que ce soit.

Questions fréquemment posées

Il y a une liste FAQ (en anglais) pour App Tamer dans la section soutien du site Web de St. Clair Software. S’il vous plaît, veuillez suivre ce lien et vérifier si nous avons déjà une réponse à votre question.

Contacter St. Clair Software

Okay, vous avez cherché en long et en large et vous n'avez pas trouvé la réponse à votre question, ou peut-être avez-vous trouvé un bogue que vous souhaiteriez nous signaler ? Ou mieux encore, peut-être souhaiteriez-vous nous dire tout le bien que vous pensez d’App Tamer, ou nous faire une suggestion intéressante. Dans tous les cas, dites-le-nous ! Vous pouvez sélectionner Assistance technique depuis le menu Aide d’App Tamer afin de vous rendre sur notre site Internet, où vous trouverez un formulaire pour nous faire parvenir un courriel.
